js中的数组类型是动态数组；

1.创建数组方式

let type = new Array()

let type = new Array(“元素1”，”元素2”….)

let type = new Array(length) length:数组长度

let type = [“元素1”，”元素2”….]

let type = []

注意：与对象一样，在使用数据字面量表示法创建数组不会调用Array构造函数

Array 构造函数还有两个 ES6 新增的用于创建数组的静态方法：from()和 of()。from()用于将类数组结构转换为数组实例，而 of()用于将一组参数转换为数组实例。
Array.from()的第一个参数是一个类数组对象，即任何可迭代的结构，或者有一个 length 属性

使用数组字面量初始化数组时，可以使用一串逗号来创建空位（hole）。ECMAScript 会将逗号之间相应索引位置的值当成空位。

ES6 新增方法普遍将这些空位当成存在的元素，只不过值为 undefined
